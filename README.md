# Chessproject
This is a 2-player Chess game implemented in C++ using the SFML library for graphics and user interaction. The game supports:

Standard chess moves for all pieces.

Drag-and-drop piece movement with mouse.

Pawn promotion (to Queen, Rook, Bishop, or Knight) for both White and Black.

Check, checkmate, and stalemate detection.

Visual display of the chessboard and piec
